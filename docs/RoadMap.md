# RoadMap

このファイルは、仕様として固定する前の保留タスク、近い将来の作業、長期構想を置くためのメモです。
詳細仕様が固まったものは、各領域の `Spec.md` や `README.md` に移します。

## しばらく

- ops 日次集計を NY Close / broker day 基準へ変更
  - 現状の `trade_date_local` は setting の `market_tz` 基準
  - ops 側で閉じる課題なので、runtime / runtime 参照 DB の安定化を優先する
  - 統合バックテストや異常検知の前に、日次境界を揃える
- 統合バックテスト
  - E011: conflict競合
  - E012: 自動昇降格シミュレーション
  - E013: 再投資50%、税金20%、手取り30%シミュレーション
- 異常検知
- AWS構成図
- configのパッキング
